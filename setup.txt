n Project Installation Guide
1. Backend Setup (Python)
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
This creates and activates a virtual environment, then installs all backend dependencies.
2. Frontend Setup (Next.js)
cd frontend
npm install
Installs all required frontend dependencies.
3. Database & Qdrant Initialization
The script backend/scripts/bootstrap_backend.py automatically handles backend initialization tasks,
including:
- Loading environment variables from your .env file (e.g., DATABASE_URL, Qdrant config)
- Creating or updating PostgreSQL tables using SQLAlchemy ORM
- Setting up and verifying the Qdrant vector collection
cd backend
python scripts/bootstrap_backend.py
This will set up all required database tables, initialize Qdrant, and print status messages for each
step.
4. Webhook Setup (Shopify Integration)
1. Get your backend domain URL using ngrok:
ngrok http 8000
Copy the generated HTTPS URL (e.g., https://abc123.ngrok.io).
2. Register Shopify Webhooks:
cd backend
python scripts/setup_webhooks.py <YOUR_URL>
This automatically creates all required Shopify webhooks pointing to your backend endpoints.
3. Manage Webhooks:
python scripts/setup_webhooks.py list
python scripts/setup_webhooks.py delete
You can list or delete webhooks as needed.
